# PETWORK API

## Description

GraphQL API with Nest to provide Petwork application. 

## Installation

```bash
yarn install
```

## Production

```bash
docker-compose build .
docker-compose up &
```

## Development

```bash
yarn run start # Basic run

yarn run start:dev # Run with hot-reload

yarn run test # Run unit tests

yarn run test:cov # Run coverage code
```
